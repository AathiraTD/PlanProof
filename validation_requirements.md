# Planning Validation Requirements

RULE 1: Site Address Validation

All planning applications must include a valid site address.

Required fields: site_address

Evidence: application_form, site_plan

Severity: error

Keywords: address, postcode, location

RULE 2: Proposed Use Validation

The proposed use or development type must be clearly stated.

Required fields: proposed_use

Evidence: application_form, design_statement

Severity: error

Keywords: use, development, proposal

RULE 3: Application Reference

Application reference must follow the standard format.

Required fields: application_ref

Evidence: application_form

Severity: warning
